# @fev/load

> 加载模块 [查看源码](https://github.com/nwa2018/fev-lib/tree/master/packages/load)

## 安装
使用`npm`
```
npm i @fev/load
```
或者使用`yarn`
```
yarn add @fev/load
```


## 使用
``` js
import { asyncLoadJs } from '@fev/load'

// 返回promise
asyncLoadJs('https://xxxxx')
```

## 命令
运行以下命令进行开发`debug`
``` js
npm run dev
// 或者
fev d --cwd dev
```
运行以下命令进行打包构建
``` js
npm run build
// 或者
fev b
```
