# @fev/fev-utils

> 工具库 [查看源码](https://github.com/nwa2018/fev-lib/tree/master/packages/fev-utils)

## 安装
使用`npm`
```
npm i @fev/fev-utils
```
或者使用`yarn`
```
yarn add @fev/fev-utils
```

## 使用
``` js
import Utils from '@fev/fev-utils'
```

xxxx

## 命令
运行以下命令进行开发`debug`
``` js
npm run dev
// 或者
fev d --cwd dev
```
运行以下命令进行打包构建
``` js
npm run build
// 或者
fev b
```
